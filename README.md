# Atom-Weaver

## Three-Stage Resonance Ladder (Cu/Pd/Ag)

This repository now captures the proposed three-layer stack behavior:

- **Silver (47) → Palladium (46)** defines the upper coupling gap (base spark/frequency).
- **Palladium (46) → Copper (29)** defines the lower high-pressure gap.
- Copper is treated as the bottom **collection/receiver plate**.

### Operational guidance

If the copper plate is used as the output tap, do **not** leave it fully “hot” in uncontrolled operation.  
Use damping/regulation (load + current limiting) so broadcast strength is controllable without runaway spikes.
